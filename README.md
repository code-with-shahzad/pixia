# Pixia

Pixia is an AI-powered application designed to provide services like image generation and, in the future, text-to-voice capabilities. It features a simple, intuitive interface with a GPT-like chat experience for generating assets.

## Features

-   **User Authentication**: Secure Login and Signup functionality.
-   **AI Chat Interface**: A simple, conversational interface (similar to GPT) to request and generate images.
-   **Profile Management**: Manage user details and generated assets.
-   **Future Expansion**: Planned support for text-to-voice and more.

## Getting Started

> **Note**: Make sure you have completed the [React Native - Set Up Your Environment](https://reactnative.dev/docs/set-up-your-environment) guide before proceeding.

### 1. Start Metro

First, you will need to run **Metro**, the JavaScript build tool for React Native.

```sh
# Using npm
npm start

# OR using Yarn
yarn start
```

### 2. Build and run your app

With Metro running, open a new terminal window/pane from the root of your React Native project, and use one of the following commands to build and run your Android or iOS app:

### Android

```sh
# Using npm
npm run android

# OR using Yarn
yarn android
```

### iOS

```sh
# Using npm
npm run ios

# OR using Yarn
yarn ios
```
